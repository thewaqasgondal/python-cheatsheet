# Basic Python Concepts

This directory contains fundamental Python programming concepts and examples.

## Files

- `data_structures.py` - Lists, tuples, dictionaries, sets
- `functions.py` - Function definitions, parameters, returns
- `decorators.py` - Function decorators and their usage
- `generators.py` - Generator functions and expressions
- `async_await.py` - Asynchronous programming with coroutines
- `context_managers.py` - Custom context managers with `__enter__` and `__exit__`
- `exception_handling.py` - Error handling patterns and custom exceptions
