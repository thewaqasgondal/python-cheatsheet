# Testing Examples

This directory contains examples for testing Python code.

## Files

- `unittest_examples.py` - Unit testing with Python's built-in unittest module
- `pytest_examples.py` - Testing with pytest framework
- `mock_examples.py` - Mocking and patching for tests
- `test_organization.py` - Best practices for organizing test code

## Prerequisites

```bash
pip install pytest
```

## Topics Covered

- **Unit Testing**: Testing individual functions and methods
- **Test Organization**: Fixtures, parametrization, test discovery
- **Mocking**: Isolating code under test with mocks and patches
- **Assertions**: Various assertion methods and patterns
- **Test-Driven Development**: Writing tests before code
