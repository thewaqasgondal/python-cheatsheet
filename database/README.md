# Database Examples

This directory contains examples for database operations in Python.

## Files

- `sqlite_basics.py` - Basic SQLite operations (CRUD, queries, schema)
- `sqlalchemy_example.py` - ORM usage with SQLAlchemy
- `database_migrations.py` - Database schema migrations and versioning

## Prerequisites

```bash
pip install sqlalchemy
```

SQLite comes built-in with Python, so no additional installation is needed for basic examples.

## Topics Covered

- **SQLite Basics**: Connection, CRUD operations, transactions
- **SQLAlchemy ORM**: Object-relational mapping, queries, relationships
- **Schema Management**: Table creation, migrations, constraints
- **Best Practices**: Connection pooling, error handling, security
