# Web Scraping Examples

This directory contains examples for web scraping and HTTP requests in Python.

## Files

- `basic_scraping.py` - Basic web scraping with requests and BeautifulSoup
- `api_requests.py` - Working with REST APIs and JSON data
- `selenium_example.py` - Browser automation with Selenium

## Prerequisites

```bash
pip install requests beautifulsoup4 lxml selenium
```

For Selenium, you'll also need a web driver (ChromeDriver, GeckoDriver, etc.).

## Topics Covered

- **HTTP Requests**: Making GET/POST requests, handling responses
- **HTML Parsing**: Extracting data from web pages with BeautifulSoup
- **API Integration**: Working with REST APIs and JSON responses
- **Browser Automation**: Automated web interaction with Selenium
- **Error Handling**: Robust scraping with proper exception handling
