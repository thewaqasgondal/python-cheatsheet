# Text Processing Examples

This directory contains examples for text processing and manipulation in Python.

## Files

- `regular_expressions.py` - Comprehensive regex patterns and usage examples
- `string_operations.py` - Advanced string manipulation techniques
- `text_analysis.py` - Text analysis and natural language processing basics

## Topics Covered

- **Regular Expressions**: Pattern matching, validation, extraction
- **String Operations**: Formatting, encoding, manipulation
- **Text Analysis**: Tokenization, frequency analysis, text statistics
